# ⭐ PROBLEM TODAY

Today your step:

```bash
gh api POST repos/org/template/generate ...
```

If repo exists →

👉 GitHub returns **422 or 409**

Then:

```
Process exited with code 1
```

So workflow stops.

---

# ✅ CORRECT ENTERPRISE APPROACH

You must introduce a **Pre-Check Step**

Flow should be:

```
1️⃣ Validate repo name format
2️⃣ Check if repo exists
3️⃣ If NOT exists → Create repo
4️⃣ If exists → Skip creation
5️⃣ Continue next steps
```

---

# ⭐ STEP 1 — Validate Repo Name

Add this step BEFORE creation.

```yaml
- name: Validate Repo Name
  run: |
    REPO="${{ github.event.inputs.repo_name }}"

    echo "Validating repo name: $REPO"

    if [[ ! "$REPO" =~ ^[a-z0-9._-]+$ ]]; then
      echo "❌ Invalid repo name"
      echo "Only lowercase letters, numbers, ., _, - allowed"
      exit 1
    fi
```

---

# ⭐ STEP 2 — Check If Repo Already Exists

This is VERY IMPORTANT step.

```yaml
- name: Check if repo exists
  id: repo-check
  env:
    GH_TOKEN: ${{ steps.generate-token.outputs.token }}
  run: |
    if gh api repos/cvs-health-source-code/${{ github.event.inputs.repo_name }} > /dev/null 2>&1; then
      echo "exists=true" >> $GITHUB_OUTPUT
      echo "Repo already exists"
    else
      echo "exists=false" >> $GITHUB_OUTPUT
      echo "Repo does not exist"
    fi
```

---

# ⭐ STEP 3 — Create Repo ONLY IF NOT EXISTS

Now use condition.

```yaml
- name: Create Repo From Template
  if: steps.repo-check.outputs.exists == 'false'
  env:
    GH_TOKEN: ${{ steps.generate-token.outputs.token }}
  run: |
    gh api \
      --method POST \
      repos/cvs-health-source-code/mics-create-repo-template/generate \
      -f owner=cvs-health-source-code \
      -f name="${{ github.event.inputs.repo_name }}" \
      -f description="${{ github.event.inputs.repo_description }}" \
      -F private=true \
      -F include_all_branches=true
```

---

# ⭐ STEP 4 — Add Friendly Log If Repo Exists

```yaml
- name: Repo already exists — skipping creation
  if: steps.repo-check.outputs.exists == 'true'
  run: |
    echo "Repository already present. Continuing provisioning steps..."
```
